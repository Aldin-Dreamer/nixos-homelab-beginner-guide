## Introduction to NixOS: Homelabbing & Sysadmin for Beginners

Welcome! This guide is designed to take you from a Linux beginner to confidently running a secure, reproducible homelab using **NixOS**. 

Unlike traditional distributions, NixOS uses a declarative configuration model, making it the ultimate tool for system administrators and self-hosting enthusiasts who value stability, security, and version-controlled infrastructure.

## What You Will Learn
* **NixOS Foundations:** Moving from imperative commands to declarative syntax.
* **Secure Homelabbing:** Deploying self-hosted services without compromising host security.
* **Sysadmin Essentials:** Managing users, networking, rollbacks, and automated backups.
* **Deep Dive Linux Concepts:** Includes a comprehensive legacy guide on manual data-at-rest encryption (structured via Arch Linux) to understand fundamental Linux security architecture.
